# Web Technologies Fundamentals Assignment

## 📌 Overview

This project demonstrates basic understanding of HTML, PHP, MySQL, and web application architecture.

## 🔧 Technologies Used

* HTML5
* PHP
* MySQL
* XAMPP
* Browser Developer Tools
* Burp Suite

## 📂 Tasks Completed

### 1. HTML Pages

* Created 5 HTML pages including forms, tables, and lists
* Implemented GET and POST methods
* Used browser developer tools for inspection and modification

### 2. PHP Scripting

* Developed 10 PHP scripts (variables, loops, functions)
* Built a simple login system
* Handled $_GET and $_POST variables
* Connected PHP with MySQL database

### 3. Web Request/Response Analysis

* Intercepted HTTP requests using Burp Suite
* Modified GET/POST parameters
* Analyzed cookies and session tokens
* Studied HTTP headers

### 4. Source Code Analysis

* Inspected 10 websites
* Identified hidden fields and comments
* Used Wappalyzer for technology detection
* Extracted form actions and validations

## 🚀 How to Run

1. Install XAMPP/WAMP
2. Place project folder in `htdocs`
3. Start Apache & MySQL
4. Open browser:

   ```
   http://localhost/web-technologies-fundamentals
   ```
